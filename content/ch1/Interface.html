常用介面

  • Hierarchy - 場景中遊戲物件列表
  • Scene - 場景編輯
  • Game - 場景預覽
  • Inspector - 物件檢視器
  • Project - 專案資料夾
  • Animation - 動畫編輯器
  • Animator - 動畫控制器
  • Console - 除錯訊息
  • Asset Store - 資源商店
  • Layer - 物件分層選單
  • Center/Pivot Settings 中點/錨點設置

Scene 場景編輯

快捷鍵

  • 畫面拖曳：Q或滑鼠中鍵
  • 畫面旋轉：Alt+滑鼠左鍵
  • 畫面縮放：Alt+滑鼠右鍵或滑鼠滾輪
  • 物件拖曳：W
  • 物件旋轉：E
  • 物件縮放：R
  • 圖像縮放：T
  • 將選定物件專注於畫面中央：選定物件後按F

Game 場景預覽

  • Free Aspect：調整遊戲畫面比例，例如：在iOS平台之下可以根據不同款裝置調整解析度。
  • Maxmize on Play：遊戲執行時將畫面放到最大。
  • Mute Audio：遊戲執行時關閉音效。
  • Stats：觀察音效、繪圖效能以及網路等數值。
  • Gizmos：顯示在編輯視窗會出現的輔助小圖示。

Inspector 物件檢視器

檢視遊戲物件中所包含的元件與屬性，同時亦可以進行編輯。

Project 專案資料夾

專案的所有素材都位於此窗格當中，包含2D圖、3D模型、腳本、場景及動畫等，並且只要用拖曳的方式便可匯入素材。要打開特定素材所在的實體資料夾位置，只需右鍵點擊該物件選Show in Explorer(Win)或Reveal in Finder(Mac)即可。

Animation 動畫編輯器

每個動畫片段包括了許多元件(Transform, Sprite)與影格，你可以透過修改屬性(position, color)來達成動畫效果。

角色動畫可以以一張圖呈現，也可以將各個關節拆解開來分別製作，當然後者在動作部分的呈現會比較細膩，但相對的所花費的時間跟人力成本也就高。

Animator 動畫控制器

Animator形容成控制物件的狀態比較貼切，像是從待命(Idle)狀態轉換到走動(Walk)狀態，其轉換條件是以動畫參數為依據。

Console 除錯訊息

所有Debug以及Error的訊息皆會詳列在此窗格，為了避免混淆個人建議勾選Clear on Play，在遊戲執行的時候清空上一次執行所產生的訊息。

Asset Store 資源商店

資源下載步驟

  1. Window > Asset Store
  2. 輸入關鍵字搜尋 (ex. 2d platformer)
  3. Download / Import
  4. 篩選需要的素材後按下Import

導入資源包

Assets > Import Package > Custom Package...

匯出資源包

  1. Assets > Export Package // 先在Project選擇要匯出的資源
  2. 直接選取物件後點擊右鍵 Export Package...
