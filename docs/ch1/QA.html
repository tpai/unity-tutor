疑難雜症

在MonoDevelop編輯器下按'('會跳出錯誤

錯誤訊息如下

System.MissingMethodException Int32 System.Environment.get_CurrentManagedThreadId()

解決方法是安裝 .NetFramework 4.5 以上的版本

Ref: stackoverflow

SDK Manager閃退

無法執行的原因是Android.bat搜尋不到java_exe的路徑，因此改為手動設定即可修正此問題。

修改前

set java_exe=
call lib\find_java.bat
if not defined java_exe goto :EOF

修改後

set java_exe=C:\Program Files\Java\jdk1.7.0_21\bin\java.exe
REM call lib\find_java.bat
REM if not defined java_exe goto :EOF

Ref: yu0410aries.blogspot.tw
