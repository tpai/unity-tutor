Google Game Service

排行榜

  • 先至Developer Console申請帳號，開啟遊戲服務功能後取得Application ID。
  • 下載Android SDK Tools
  • 將Preferences中Enternal Tools分頁的Android SDK Location路徑指定至上述檔案解壓縮後的資料夾
  • 利用Android SDK Manager下載Google Play Services Package
  • File -> Play Games - Andriod Setup 填入Application ID
  • 語法簡介如下所示： 
void Start () {
    // 顯示debug log
    PlayGamesPlatform.DebugLogEnabled = true;
    // 初始化Play Game服務
    PlayGamesPlatform.Activate();
}
void SignIn () {
    Social.localUser.Authenticate((bool success) => {
        if(success) { /* 登入成功 */ }
        else { /* 登入失敗 */ }
    });
}
void SubmitScore () {
    Social.ReportScore(score, game-service-id, (bool success) => {
        if(success) { /* 分數更新成功 */ }
        else { /* 分數更新失敗 */ }
    });
}
void ShowLeaderboard () {
    // 顯示排行榜
    Social.ShowLeaderboardUI();
}
  • 新增應用程式，並將APK上傳至Alpha測試階段。
  • 再到遊戲服務分頁中連結剛上傳的App後，即可測試排行榜功能。
